Project Darwinism
Status: Version 1.1.0 - Biological Overhaul

Project Darwinism is a high-yield serum designed for rapid field adaptation. This isn't just a potion; it's a genetic rewrite that monitors the host's physical state and environment to trigger localized mutations.

Current Adaptations
- **Kinetic Overdrive:** Sprinting for 3s triggers a massive velocity spike (Speed II).
- **Survival Instinct:** Below 1 heart of health, the host enters an invulnerable state (Resistance V / Regen V).
- **Emergency Metabolism:** Near-starvation triggers a 5% chance of spontaneous calorie saturation.
- **Efficiency Evolution:** Detecting repetitive impact against solid blocks triggers Haste II.
- **Optic Adaptation:** Automatic Night Vision in light levels < 2.

Technical Implementation
This mod utilizes **Persistent Player Variables** to track adaptation timers across game ticks, ensuring biological memory isn't lost during server lag or player movement.
